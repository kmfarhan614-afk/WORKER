## 📸 HOW TO REPLACE GALLERY IMAGES

### Step 1: Prepare Your Photos
```
Take high-quality photos of:
1. Hair cutting service
2. Beard styling service
3. Hair coloring service
4. Keratin treatment
5. Hydra facial treatment
6. Hair styling service

Recommended size: 300x300 pixels
File format: JPG or PNG
File size: < 50KB each
```

### Step 2: Optimize Images
```
Use free tools:
- TinyPNG: https://tinypng.com
- Squoosh: https://squoosh.app
- ImageOptim: https://imageoptim.com

Target: < 50KB per image
Quality: High resolution
Format: JPG for photos
```

### Step 3: Replace in HTML
```html
<!-- Find this line in index.html -->
<img src="https://via.placeholder.com/300x300/1a1a1a/d4af37?text=Professional+Hair+Cut" alt="Professional Hair Cut">

<!-- Replace with your image -->
<img src="hair-cutting.jpg" alt="Professional Hair Cut">
```

### Step 4: Upload Files
```
When deploying:
1. Upload index.html
2. Upload all 6 gallery images
3. Make sure images are in same folder
4. Test website to verify images display
```

## 💡 PHOTO TIPS FOR GALLERY

### Lighting
- Use natural light when possible
- Avoid harsh shadows
- Take photos during daytime
- Use professional lighting if available

### Composition
- Frame the service being performed
- Show the result clearly
- Include before & after if possible
- Keep background clean and professional

### Quality
- Use high-resolution camera or phone
- Take multiple shots
- Choose the best ones
- Ensure photos are sharp and clear

### Consistency
- Use similar lighting for all photos
- Maintain consistent style
- Keep color tones similar
- Professional appearance throughout
