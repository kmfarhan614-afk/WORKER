## 📋 HOW TO REPLACE IMAGES

### Step 1: Prepare Your Photos
```
1. Take high-quality photos of:
   - Your salon interior
   - Hair cutting services
   - Beard styling
   - Hair coloring
   - Keratin treatment
   - Hydra facial
   - Hair styling

2. Recommended sizes:
   - About section: 500x400 pixels
   - Gallery images: 300x300 pixels
   - Hero background: 1920x1080 pixels (already done)

3. File formats:
   - JPG (best for photos)
   - PNG (for graphics)
   - WebP (for web optimization)
```

### Step 2: Optimize Your Photos
```
Use free tools to optimize:
- TinyPNG: https://tinypng.com
- Squoosh: https://squoosh.app
- ImageOptim: https://imageoptim.com

Target sizes:
- About image: < 100KB
- Gallery images: < 50KB each
- Hero background: < 200KB
```

### Step 3: Replace in Website

#### Option A: Using Local Files
```html
<!-- Replace this line in index.html -->
<img src="https://via.placeholder.com/500x400/1a1a1a/d4af37?text=Salon+Interior" alt="New Trimmer Hair Salon Interior">

<!-- With this line -->
<img src="salon-interior.jpg" alt="New Trimmer Hair Salon Interior">
```

#### Option B: Using Online URLs
```html
<!-- Replace with your image URL -->
<img src="https://your-image-hosting.com/salon-interior.jpg" alt="New Trimmer Hair Salon Interior">
```

### Step 4: Upload Files
```
When deploying to hosting:
1. Upload index.html
2. Upload all image files
3. Make sure images are in same folder as index.html
4. Test website to verify images display
```

## 📸 PHOTO TIPS FOR BEST RESULTS

### Lighting
- Use natural light when possible
- Avoid harsh shadows
- Take photos during daytime
- Use professional lighting if available

### Composition
- Frame the subject well
- Show the service being performed
- Include before & after if possible
- Keep background clean and professional

### Quality
- Use high-resolution camera or phone
- Take multiple shots
- Choose the best ones
- Ensure photos are sharp and clear

### Consistency
- Use similar lighting for all photos
- Maintain consistent style
- Keep color tones similar
- Professional appearance throughout
