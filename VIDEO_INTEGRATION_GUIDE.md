## 🎬 VIDEO OPTIMIZATION TIPS

### File Size Optimization
- Use video compression tools
- Target: 20-50MB for web
- Tools: HandBrake, FFmpeg, Adobe Media Encoder

### Quality vs Size
- 1080p: Best quality, larger file
- 720p: Good quality, smaller file
- 480p: Lower quality, very small file

### Recommended Settings
- Resolution: 720p or 1080p
- Bitrate: 2-5 Mbps
- Format: MP4 (H.264)
- Audio: AAC, 128kbps
